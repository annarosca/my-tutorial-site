Wow! Such code snippets!
Let's see another, with line highlighting:

```js highlights=3-5
// fizzbuzz in JS
for (let i = 1; i <= 100; i++) {
    let out = '';
    if (i % 3 === 0) out += 'Fizz';
    if (i % 5 === 0) out += 'Buzz';
    console.log(out || i);
}
```
